# RazkarStudio Licenses

Hello! This repository holds all licenses made and published by RazkarStudio, in a reusable template.

> A common trait of the RazkarStudio licenses are that they forbid deliberate AI/ML training use.

The following are the published licenses:
- [RSGL-v1.0](./RSGL-v1.0.md): RazkarStudio Game License. Source-available license for games.
- [RSPL-v1.0](./RSPL-v1.0.md): RazkarStudio Permissive License. Alike open-source, but with AI/ML limitations and patent grants.

Thanks for checking this repo out.
