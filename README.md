# RazkarStudio Licenses

Hello! This repository holds all licenses made and published by RazkarStudio, in a reusable template.

> A common trait of the RazkarStudio licenses are that they forbid deliberate AI/ML training use.

The following are the published licenses:
- [RSGL-v1.0](./RSGL-v1.0.md): RazkarStudio Game License. Source-available license for games.
- [RSPL-v1.0](./RSPL-v1.0.md): RazkarStudio Permissive License. Alike open-source, but with AI/ML limitations and patent grants.
- [RSAL](./RSAL/): RazkarStudio Source-Available License. Family of source-available licenses.
    - [RSAL-NC-v1.0](./RSAL/RSAL-NC-v1.0.md): Non-Commercial. Weak copyleft, non-commercial only distribution.
    - [RSAL-Free-v1.0](./RSAL/RSAL-Free-v1.0.md): Free. Weak copyleft, any distribution allowed.
    - [RSAL-Strict-v1.0](./RSAL/RSAL-Strict-v1.0.md): Strong copyleft, non-commercial only distribution.
    - [RSAL-View-v1.0](./RSAL/RSAL-View-v1.0.md): No distribution allowed, view-only.

Thanks for checking this repo out! Cheers, RazkarStudio.
